Trabajo Practico de Programacion en Capas

Consigna:

Realizar un pequeño programa bajo el patroón de programación en capas que debera cumplir los siguientes requisitos:

  • User Interface:

    Constará de un formulario en el cual se ingresará un numero de CUIL sin guinones.

  • Bussiness Layer:

    Deberá calcular el digito verificador del CUIL ingresado.

  • Data Layer:

    Simplemente recibirá el numero de CUIL ingresado en el formulario y le sumará 1

Funcionamiento

En el formulario de la UI se ingresará el numero de CUIL actual, restandole al ultimo digito(Codigo verificador) una unidad (*). La Bussines Layer recibira el dato, calculará el código verificador real del cuil y le enviará a la Data Layer el valor que recibió en primer instancia. Con el valor recibido, la Data Layer unicamente le sumará una unidad(N+1) y le devolverá a la Bussiness Layer el valor calculado. Una vez realizado todo este proceso, la Bussiness evaluará el valor devuelto de la Data Layer contra el CUIL con el dígito verificador creado. En caso de ser iguales, se mostrará una leyenda indicando que el CUIL es correcto, de manera contraria, incorrecto.

(*)Ej: Si tu numero real de CUIL es 20XXXXXXXXXN deberás ingresar 20XXXXXXXX(N-1)
